Projects

HPLtoPIN

HPLtoPIN was my first major coding project. In my work at Bonanza.com, I was tasked with posting lists of products to Pinterest. I was learning to code with Python and figured I could simplify this task. This app uses BeautifulSoup to grab all the listings from a Hand Picked List on Bonanza given a URL. It then creates a board on Pinterest using the name of the list, and pins each product to that newly created board. The board creation and pinning is done through the Pinterest API.

Try it out for yourself!
